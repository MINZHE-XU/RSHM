# RSHM
Hello, World! app in React and Redux.

Now using `create-react-app`

## Run the app

```
git clone https://github.com/vaibhavmule/RSHM.git

cd RSHM
npm install
npm start


If you are using Yarn

cd RSHM
yarn install
yarn start

open http://localhost:3000/

```
